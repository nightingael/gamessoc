Week 0: Drafts Are Fun

Hello World! This email acts as a template for the new Gamessoc emails, but it also provides you with a guide.
This email assumes that you have no HTML experience, so don't worry! The template should hopefully be easy to follow.
Here's a quick list of the topics we'll be covering:

  • Finding the template
  • Setting up a draft
  • Download YAMM
  • Configuring YAMM & Sending

Finding the template

The template for the emails is hosted on Github. This should link directly to the template as plaintext. If that doesn't work, ask for the Github repository for gamessoc and the template should be in the email folder.
Right, that was step one-- not so bad!

Setting up a draft

With your template downloaded, it's time to start drafting! Write your email separately and then get it approved by the committee, and then pop it into the draft using Notepad or any other simple text editor. Simple enough, paragraphs go within the p tags, your topic titles go in the h2 and li tags. If you need more paragraphs just copy/paste a h2-p set and a li set. If you want to add a link, follow the formatting that the Unsubscribe link at the end uses. When you're done, get onto Gmail, copy and paste all the code into Gmail and save it as a draft (with your email titled)!

Download YAMM

YAMM stands for Yet Another Mail Merge, a Google Sheets plugin that allows you to send fancy emails. Download it here.
You can then open a new Google Sheets document, add "announce-gamesoc@lists.redbrick.dcu.ie" to one cell of the document, and then start the mail merge. Go to the "Add-ons" tab, select Yet Another Mail Merge, and then click "Start Mail Merge".

Configuring YAMM & Sending

Nearly there now! From this page, you can choose what name to send the email as. Select your draft from earlier and you're ready! There are more configuration options, though. You can track whether people have opened the email or not (very cool!), as well as changing the reply-to address (and send-from address, but I have yet to get the gamessoc email enabled on Gmail). You can send a test email to yourself just to ensure everything is functioning properly, or send the email to the list directly. You're done!

So that's all there is to it! And it looks a little nicer than plain text.
